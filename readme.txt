To run the code for the Interactive Calculator and Shape Visualizer, you will need to install the following Python packages using pip:

1. PyQt5: This package is required for creating the GUI interface.
   ```
   pip install PyQt5
   ```

2. Matplotlib: This package is used for plotting shapes in the visualization window.
   ```
   pip install matplotlib
   ```

Ensure you have these packages installed before executing the code to ensure smooth functionality of the application. If you encounter any issues during installation or execution, feel free to ask for further assistance.
